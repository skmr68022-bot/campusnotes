Golden Rules — Must Memorize ALL UNITS
📌
Rule 1: Every adjustment has TWO effects — one in P&L/Trading and one in Balance Sheet.
📌
Rule 2: Trial Balance items go to ONLY ONE place (Trading, P&L, or Balance Sheet).
📌
Rule 3: PDD calculated AFTER writing off bad debts.
📌
Rule 4: Discount on Debtors calculated AFTER PDD.
📌
Rule 5: Change in Estimate = Prospective (forward only). Change in Policy = Retrospective (restate past).
📌
Rule 6: CGST and SGST can NEVER be set off against each other.
Common Mistake Alert

Don't forget: In Final Accounts, if you only record one side of an adjustment, your Balance Sheet will NOT tally! Always check: Debit effect + Credit effect = Complete entry.

Unit 1 · Theoretical Framework UNIT 1
Memory Tricks

How to Solve Provisions & Contingencies
  1. Identify — Is it a present obligation or possible obligation?
  2. Probability Check — ≥50% = Probable → Provision; <50% = Possible → Contingent Liability
  3. Reliability Check — Can you estimate the amount reliably? If yes → Provision; If no → Contingent Liability
  4. Treatment — Provision = Recognized (Debit P&L, Liability in B/S); Contingent = NOT recognized (Disclose in Notes)
Quick Recall
"Provisions are Probable & Reliable — Contingent are Possible & Unreliable"

Ind AS Applicability: Listed + Unlisted (Net Worth ≥ ₹250 Cr) → Ind AS · Others → Traditional AS

Unit 2 · Depreciation & Inventory UNIT 2
📐
SLM Depreciation: (Cost – Scrap) × Rate % → Same every year
📐
WDV Depreciation: Opening WDV × Rate % → Decreases every year
📅
Depreciation Rule: Bought before 1st April = Full year · After 1st October = Half year
📦
FIFO: Oldest stock sold first → Closing stock from latest purchases
⚖️
Weighted Average: Recalculate average after EVERY purchase
🔥
Normal Loss: Reduce quantity only · Abnormal Loss: Remove quantity & cost → Charge to P&L
How to Solve Change in Depreciation Method
  1. Step 1: Calculate depreciation under OLD method for all years.
  2. Step 2: Recalculate depreciation under NEW method from Day 1 (Retrospective).
  3. Step 3: Compare cumulative totals → Shortfall = Debit to P&L; Excess = Credit to P&L.
  4. Step 4: In the year of change, adjust the difference in P&L.
Quick Recall
"FIFO = First In First Out · WDV = Written Down Value · SLM = Same Every Year"

AS 2: Inventory = Lower of Cost or NRV · AS 10: PPE Cost = Price + Freight + Customs + Installation

Common Mistake Alert

Don't forget: In WDV method, depreciation is calculated on Opening WDV — NOT on cost! And in FIFO, always sell the oldest stock first.

Unit 3 · Final Accounts & NPOs UNIT 3
📊
COGS: Opening Stock + Purchases + Direct Expenses – Closing Stock
📊
Gross Profit: Sales – COGS
📊
Net Profit: GP + Other Incomes – Other Expenses
📊
Balance Sheet: Assets = Liabilities + Capital
📋
Closing Capital: Opening Capital + Net Profit + Interest on Capital – Drawings – Interest on Drawings
📋
NPO Subscription: Received – Outstanding(beg) – Advance(end) + Outstanding(end) + Advance(beg)
How to Solve Final Accounts (Step-by-Step)
  1. Step 1: Read the Trial Balance and mark all adjustments (star them!).
  2. Step 2: Draw Trading A/c, P&L A/c, and Balance Sheet formats side by side.
  3. Step 3: One-touch rule — Post each Trial Balance item to ONLY ONE place.
  4. Step 4: For adjustments, use the inner column — show adjustment in inner column, net in outer column.
  5. Step 5: Always record TWO effects for each adjustment (one in P&L/Trading, one in Balance Sheet).
  6. Step 6: Check if totals match! If not, run the diagnostic check.
Quick Recall — Adjustments
"Closing Stock = Credit to Trading + Asset · Outstanding = Add to Expense + Liability · Prepaid = Subtract from Expense + Asset"

PDD: Calculate AFTER bad debts · Discount on Debtors: Calculate AFTER PDD

Common Mistake Alert

Don't forget: In PDD, Debtors amount = Sundry Debtors – Further Bad Debts. In Discount on Debtors, amount = Debtors after PDD.

Unit 4 · Branches, Departments & Leases UNIT 4
🏪
Debtors System — Branch Account: Debit = Opening Stock + Goods sent + Expenses · Credit = Sales + Cash from Debtors + Closing Stock
📐
Closing Debtors: Opening + Credit Sales – Cash – Discount – Bad Debts
🏷️
Loading (Cost+25%): 1/5 of Invoice Price · Loading (Cost+50%): 1/3 of Invoice Price
🏢
Departmental — Rent: Floor Area · Salaries: Employees · Depreciation: Asset Value · Selling: Sales
📄
GIL: (Rental × Years) + GRV + UGRV
📄
UFI: GIL – Fair Value · Net Investment: Fair Value – UGRV
How to Solve Stock & Debtors System (with Loading)
  1. Step 1: Find Loading % — Cost+25% = 1/5 of IP; Cost+50% = 1/3 of IP.
  2. Step 2: Prepare Branch Stock A/c — all entries at Invoice Price.
  3. Step 3: Prepare Branch Debtors A/c — find closing debtors.
  4. Step 4: Calculate Stock Reserve = Closing Stock × Loading %.
  5. Step 5: Prepare Branch Adjustment A/c — remove loading from Goods sent, Returns, Stock, Abnormal Loss → finds Gross Profit.
  6. Step 6: Prepare Branch P&L A/c — Gross Profit – Expenses = Net Profit.
Quick Recall
"Cost+25 = 1/5 IP · Cost+50 = 1/3 IP · GIL = Total Money · UFI = Profit/Interest"

Branch Profit/Loss: Credit side bigger = Profit · Debit side bigger = Loss

Common Mistake Alert

Don't forget: In Stock & Debtors System, Stock Reserve is calculated on Closing Stock only! And in Branch Debtors A/c, Credit Sales comes from Branch Stock A/c.

How to Solve GST — ITC Utilization
  1. Step 1: Start with Input IGST → set off against Output IGST FIRST.
  2. Step 2: Use remaining Input IGST against Output CGST, then Output SGST.
  3. Step 3: Use Input CGST → set off against Output CGST FIRST, then Output IGST.
  4. Step 4: Use Input SGST → set off against Output SGST FIRST, then Output IGST.
  5. Step 5: REMEMBER: CGST and SGST can NEVER be set off against each other!
  6. Step 6: Cash Payable = Total Output GST – Total ITC Utilized.
Quick Recall — GST Order
"IGST can go anywhere (IGST → CGST → SGST). CGST can go to CGST or IGST. SGST can go to SGST or IGST. CGST and SGST can NEVER swap with each other!"

Common Mistake Alert

Don't forget: Input CGST CANNOT be set off against Output SGST! And Input SGST CANNOT be set off against Output CGST! This is the BIGGEST GST trap in exams.

Diagnostic Checklist — If Balance Sheet Doesn't Tally ALL UNITS
🔍
Check 1: Did you bring down Net Profit/Loss from P&L to Balance Sheet? (Most common mistake!)
🔍
Check 2: Did you record TWO effects for every adjustment? (One in P&L/Trading, one in B/S)
🔍
Check 3: Did you calculate PDD correctly? (After bad debts, not before)
🔍
Check 4: Did you calculate Discount on Debtors correctly? (After PDD, not before)
🔍
Check 5: Did you deduct depreciation from the correct asset in Balance Sheet?
🔍
Check 6: Did you add outstanding expenses and subtract prepaid expenses correctly?
Exam-Day Speed Trick

Inner Column Method: When solving adjustments, write the adjustment in an inner column and the net amount in the outer column. This saves time and reduces errors!

Example: Salaries ₹2,20,000 + Outstanding ₹20,000 = Net ₹2,40,000
