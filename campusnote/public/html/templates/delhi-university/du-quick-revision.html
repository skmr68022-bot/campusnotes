How to Use This Quick Revision

This file is made for fast revision before Delhi University semester exams. It should be used after completing full notes. The aim is to revise the most scoring concepts quickly without reading long explanations again.

Best Use

Use this file 1–2 days before the exam to revise definitions, important points, difference tables, expected questions, and answer-writing structure.

Exam Strategy

First revise definitions, then revise tables and key points, then practice expected short and long answer questions.

9. Answer Writing Formats

For 3–4 Marks

  • Write direct definition or short introduction.
  • Add 3–4 clear points.
  • Add one example if possible.

For 6 Marks

  • Start with introduction.
  • Add definition.
  • Write 5–6 points under headings.
  • Add table/example if needed.
  • End with short conclusion.

For Long Answer

  • Start with a strong introduction.
  • Explain the topic with headings and subheadings.
  • Add definitions, examples, tables, or case points.
  • End with a clear conclusion.

Best DU Format

Introduction → Definition → Points/Table → Example → Conclusion.

10. Common Mistakes Before Exam

  • Do not memorize only definitions without understanding meaning.
  • Do not write long paragraphs without headings.
  • Do not skip examples in long-answer questions.
  • Do not confuse similar concepts in difference questions.
  • Do not ignore PYQ-based topics.
  • Do not write irrelevant points just to increase answer length.

13. Expected Questions

Very Short Answer Questions

  1. Define the main concept.
  2. State any two features of the topic.
  3. Give one example related to the topic.

Short Answer Questions

  1. Explain the meaning and importance of the topic.
  2. Write a short note on an important subtopic.
  3. State the features or essentials of the concept.

Long Answer Questions

  1. Discuss the concept in detail with suitable examples.
  2. Differentiate between two related concepts.
  3. Explain the features, importance, and limitations of the topic.
