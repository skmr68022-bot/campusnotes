14. How to Write Answers in DU Exams

This section improves the premium value of the notes because students learn not only the content, but also how to present answers properly.

For 3–4 Marks Questions

  • Write a direct definition or short introduction.
  • Add 3–4 relevant points.
  • Use one small example if possible.

For 6 Marks Questions

  • Start with introduction or definition.
  • Write 5–6 points under headings.
  • Add table/example/diagram if required.
  • End with a short conclusion.

For Long Answer Questions

  • Write a strong introduction.
  • Explain the concept with headings and subheadings.
  • Use examples, case points, tables, or formulas.
  • Conclude the answer in 2–3 lines.

Best DU Answer Format

Introduction → Definition → Explanation → Points/Table → Example → Conclusion.

15. Common Mistakes to Avoid

  • Writing only definitions without explanation.
  • Skipping examples in long-answer questions.
  • Using long paragraphs without headings.
  • Confusing two similar concepts in difference questions.
  • Not writing a conclusion in long answers.
  • Not connecting the answer with the exact question asked.
  • Writing irrelevant points just to increase answer length.
