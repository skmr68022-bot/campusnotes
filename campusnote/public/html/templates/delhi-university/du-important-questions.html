How to Use These Important Questions

This file is designed to help students revise exam-oriented questions before Delhi University semester exams. It includes very short, short, long, case-based, and PYQ-style questions with answer hints and priority levels.

Best Use

First revise the high-priority questions, then prepare medium-priority questions, and finally use the answer hints to improve presentation.

Exam Strategy

Do not memorize answers blindly. Understand the structure of each answer: introduction, definition, key points, example, and conclusion.

3. Very Short Answer Questions

These questions are useful for 1–2 mark answers, quick definitions, and basic concept checks.

Question Answer Hint Priority
Define the main concept. Write a direct definition in 2–3 lines. High
State any two features of the topic. Write two clear points with keywords. High
Give one example related to the concept. Use a simple practical or subject-based example. Medium
Write the meaning of an important term. Write simple meaning, not a long explanation. Medium

4. Short Answer Questions

These questions are usually suitable for 3–4 marks. Answers should be direct, structured, and supported with examples where possible.

Short Answer Format

Definition / Introduction → 3 to 4 points → Small example.

  1. Explain the meaning and importance of the main concept.
    Hint: Write definition, then 3–4 importance points.
    Priority: High
  2. Write a short note on an important subtopic.
    Hint: Write meaning, features, and one example.
    Priority: High
  3. State the essential elements / features / characteristics of the topic.
    Hint: Write points in headings with brief explanation.
    Priority: High
  4. Explain the role or relevance of the concept in practical life.
    Hint: Add one practical example.
    Priority: Medium

5. Long Answer Questions

These questions are suitable for 6 marks or higher. Answers should be written with headings, examples, tables, and conclusion.

Long Answer Format

Introduction → Definition → Detailed explanation → Features / points → Example / table → Conclusion.

  1. Discuss the concept in detail with suitable examples.
    Answer Hint: Start with definition, explain nature and importance, add 5–6 points, give example, then conclude.
    Priority: High
  2. Explain the features, importance, and limitations of the topic.
    Answer Hint: Divide answer into three headings: features, importance, limitations.
    Priority: High
  3. Describe the process / steps / essentials related to the topic.
    Answer Hint: Use numbered points and explain each step briefly.
    Priority: Medium
  4. Critically examine the concept with examples.
    Answer Hint: Write meaning, advantages, limitations, and conclusion.
    Priority: Medium

6. Important Difference Questions

Difference questions are scoring because they are easy to present in table form.

Question Important Basis Priority
Differentiate between Concept A and Concept B. Meaning, nature, purpose, example. High
Distinguish between Topic 1 and Topic 2. Scope, legal effect, practical use. Medium
Compare two related principles / formats / methods. Definition, application, limitation. Medium

Difference Answer Tip

Always use table format. Write at least 4–5 clear bases of distinction.

7. Case-Based / Application-Based Questions

Use this section for subjects where case application is important, such as Business Laws, Management, Accounting, Economics, Taxation, and practical subjects.

Case Question Format

Write a short case situation here. The student should identify the concept, apply the rule or principle, and write the conclusion.

  1. A practical situation is given. Identify the relevant concept and explain it.
    Hint: Concept → Rule / Principle → Application → Conclusion.
    Priority: High / Medium
  2. Apply the topic to a business / legal / accounting / economic situation.
    Hint: Explain the relevant provision or concept before conclusion.
    Priority: Medium

9. Marks-Wise Answer Structure

Marks Answer Length Best Structure
1–2 Marks 2–4 lines Direct definition or point-based answer.
3–4 Marks Half page Definition + 3–4 points + example.
6 Marks One page Introduction + definition + 5–6 points + conclusion.
Long Answer 1.5–2 pages Intro + detailed headings + table/example + conclusion.

10. Premium Answer Writing Hints

  • Use headings and subheadings to make answers readable.
  • Start long answers with a short introduction.
  • Write definitions in clear and accurate language.
  • Use tables for difference questions.
  • Add examples in long answers wherever possible.
  • End long answers with a short conclusion.
  • Avoid writing irrelevant points just to increase length.

Best Presentation Formula

Definition → Explanation → Points/Table → Example → Conclusion.

11. Final Revision Question List

Revise First

  1. Most expected definition question.
  2. Most important short note.
  3. Most repeated long answer.
  4. Most important difference table.

Revise Second

  1. Medium-priority short answer.
  2. Case-based application question.
  3. Features / essentials / types question.

Revise if Time Remains

  1. Low-priority backup questions.
  2. Extra examples or less repeated areas.

12. Self-Test Before Exam

Try answering these without looking at the notes.

  1. Can I define the main concept clearly?
  2. Can I write 5 points for the main long-answer question?
  3. Can I prepare one difference table?
  4. Can I write one short note in 5 minutes?
  5. Can I add an example in my answer?
  6. Can I conclude the answer properly?
