9. How to Use Formulas in Exam Answers

  • Write the formula before substituting values.
  • Mention the meaning of important terms if required.
  • Show calculations step by step.
  • Write final answer clearly with unit, ₹, %, ratio, or label.
  • For accounting questions, show working notes separately.
  • For theory answers, use formulas/rules only where relevant.

Marks Tip

Formula + correct substitution + working notes + final answer gives better chances of scoring marks.

10. Common Formula Confusions

Confusion Correct Understanding How to Avoid Mistake
Confusing Formula A with Formula B Formula A is used for one situation, Formula B for another. Check what the question is asking.
Using opening value instead of closing value Read the question carefully. Underline given values before solving.
Forgetting percentage / unit Final answer must include correct unit or symbol. Check answer label at the end.
