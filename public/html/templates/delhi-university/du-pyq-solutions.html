How to Use These PYQ Solutions

This file is designed to help students understand how Delhi University questions are asked and how answers should be written in the semester exam. Do not only memorize the answer. Study the pattern, keywords, structure, and conclusion.

Best Use

First read the question carefully, identify the topic, understand the answer structure, and then practice writing the answer in your own words.

PYQ Strategy

Repeated PYQ areas are highly important. If a topic appears repeatedly in different forms, prepare it as a full answer, short note, and difference table.

3. PYQ Answer Writing Format

Use this structure while writing PYQ answers in Delhi University exams.

Question Type Best Answer Structure What to Avoid
Definition Question Definition → Simple explanation → Example. Writing only one line without explanation.
Short Note Introduction → 4–5 points → Example. Writing long paragraph without headings.
Long Answer Intro → Definition → Headings → Table/example → Conclusion. Skipping conclusion and examples.
Difference Question Table with basis of difference. Writing difference in paragraph form.
Case-Based Question Concept → Rule/principle → Application → Conclusion. Writing conclusion without applying the concept.

8. Common Mistakes in PYQ Answers

  • Writing memorized content without understanding the question.
  • Ignoring the marks and writing too much or too little.
  • Skipping definition in concept-based questions.
  • Not using table format for difference questions.
  • Not applying the rule properly in case-based questions.
  • Forgetting conclusion in long answers.
  • Writing irrelevant points just to increase answer length.
